Sobre Nosotros

¿Quiénes Somos?

Somos una API pública dedicada a ofrecer acceso a información relacionada con los personajes y elementos del universo de *Little Witch Academia*. Nuestra API proporciona datos detallados sobre los personajes, su relación y estadísticas clave, permitiendo a los desarrolladores integrar fácilmente esta información en sus aplicaciones y proyectos.

Actualmente, contamos con una función de biblioteca interactiva en progreso, la cual permitirá explorar de forma dinámica la información sobre los personajes, sus vínculos y las estadísticas asociadas. Este desarrollo está en marcha y pronto ofrecerá una experiencia más fluida e interactiva para nuestros usuarios.

A continuación, puedes acceder a nuestras API públicas para obtener información específica a través de los siguientes enlaces:

  • Lista de personajes: Lista de las Brujas
  • Relación entre personajes: Relacion entre las Brujas
  • Estadísticas de personajes: Estadisticas de las Brujas
  • Hechizos de personajes: Hechizos de las Brujas
